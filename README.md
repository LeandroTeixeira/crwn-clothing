## Description
Full Stack E-Commerce App built with React, using Firebase, deployed with Netlify. Currently hosted at https://superb-meerkat-5c8e8c.netlify.app/

## Technologies Used
* React
* Context API
* Hooks
* Styled-Components
* Firebase API
